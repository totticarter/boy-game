# 贪吃蛇游戏 - 微信小程序版

这是一个基于微信小程序开发的贪吃蛇游戏，支持单人模式和双人对战模式。

## 功能特点

### 游戏模式
- **单人模式**：一条蛇，挑战自我，追求最高分
- **双人模式**：两条蛇同场竞技，2分钟对战

### 游戏玩法
- 🟢 **小虫子**：吃到后蛇身+1节，得10分
- 🔴 **大虫子**：吃到后蛇身+2节，得20分
- 💥 **碰到边缘**：蛇身-3节（不扣分）
- ⏱️ **游戏时长**：2分钟倒计时

### 游戏特色
- 触摸按钮控制，适配小程序操作
- 单人模式支持最高分记录
- 双人模式自动判定胜负
- 碰撞自动转向，避免游戏结束
- 震动反馈，增强游戏体验

## 如何运行

1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

2. 打开微信开发者工具，选择"导入项目"

3. 选择 `wechat-miniprogram` 文件夹作为项目目录

4. AppID选择"测试号"或使用自己的AppID

5. 点击"导入"，即可在开发者工具中预览和调试

## 开发说明

### 技术栈
- 微信小程序原生框架
- Canvas 2D 绘图
- 小程序振动API

### 核心功能
- Canvas绘制游戏画面
- 定时器控制游戏循环
- 触摸按钮控制蛇的移动
- 本地存储记录最高分

### 控制说明
- 单人模式：使用屏幕底部的方向按钮控制
- 双人模式：
  - 玩家1使用左侧按钮（↑↓←→）
  - 玩家2使用右侧按钮（WASD）
